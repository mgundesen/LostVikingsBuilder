This reporsitory is made to allow creating Lost Vikings levels. None of the code uses the original sources and are made to imitate the original game (see "changes_from_original.txt on what changes). This allows using Godot's build in level editing tools and make new components with less effort compared to rom hacking. Resulting way to play the game is an executeable generated by the Godot system.

The currently released binaries are my own level pack using the level building code.

In case you want to create your own levels using the system you will need :
- Godot, currently 4.5.1 the current stable at time of writing (https://godotengine.org/)
- Source code in this repository
