This reporsitory is made to allow creating Lost Vikings levels. This is done by adding your own level pack in folder "Levels". Easiest start to a level is making a duplicate of template and start modyfing it directly from the editor in Godot (https://godotengine.org/)

There still need to be created a more flexible system to define the level list, which is currently in "scene_control.gd"
